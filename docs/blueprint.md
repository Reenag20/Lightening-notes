# **App Name**: Zenith Notes

## Core Features:

- Lightning Capture: Single-field quick add with auto-generated title and keyboard-first navigation.
- Brutal Design Language: Stark black/white contrast and raw grid system for maximum clarity and information density.
- Smart Organization: Tag-based taxonomy with full-text search and auto-suggested tags tool for efficient note management.
- Lean Authentication: Email/password authentication with persistent sessions and anonymous mode.
- Author Dashboard: Split view for note editing and list management, with usage stats and export options.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) to evoke focus and intellect.
- Background color: Very light gray (#F0F2F5) to provide stark contrast.
- Accent color: Electric Blue (#7DF9FF) to highlight key actions and elements.
- Headline font: 'Space Grotesk' sans-serif for a computerized feel.
- Body font: 'Inter' sans-serif for a clean, modern look.
- Minimalist line icons for common actions and tags.
- Dense information architecture maximizing screen real estate.
